### React App
- **Description**: A basic React application with Docker setup.
- **Files**:
  - `Dockerfile`
  - `package.json`
  - `src/`, `public/`
- **How to Run**:
  ```bash
  docker build -t react-app .
  docker run -p 3000:3000 react-app
  ```

- **Open your web browser and navigate to `http://localhost:8080` to view the react page.**
